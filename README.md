# Step Count Tracker (VBA-Based Excel System)

## About
A **VBA-powered Excel system** for capturing and analyzing step count data, with **automated performance rating** and **reporting features**.  
It helps track daily step counts, calculate performance metrics automatically, and generate summary reports for analysis.

## Features
- Log daily step count per employee/user
- Automated performance ratings based on step targets
- Generate summary reports and charts automatically
- Conditional formatting to highlight high/low performers
- Macro buttons for adding data, updating reports, and refreshing dashboards
- Easy-to-customize targets and scoring criteria

## Technologies Used
- Microsoft Excel (macro-enabled `.xlsm` file)
- VBA (Visual Basic for Applications) for automation
- Excel formulas for calculations and reporting
